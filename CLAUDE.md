Academic website for Dr. Shannon Saszik (NEIU Psychology). Static single-page site with automated citation fetching.

**Stack:** Vanilla HTML5 + CSS (custom properties) + ES modules — no framework, no build step.

**Hosting:** GitHub Pages serves repo root directly.

## Development

```bash

python3 -m http.server 8000
# Open http://localhost:8000
```

The site uses ES modules and `fetch('./data/citations.json')`, so it must be served over HTTP, not opened as file://.

## Architecture

```
index.html          # Single-page site with all sections
assets/
  css/style.css     # All styles — mobile-first, CSS custom properties
  js/
    theme.js        # Dark/light mode — runs in <head> pre-paint to avoid FOUC
    main.js         # Nav, scroll, animations, mobile menu, copy-email
    citations.js    # Fetches and renders data/citations.json
data/
  config.json       # Research keywords, profile info, citation fetch params
  citations.json    # Auto-generated by GitHub Actions; do not edit manually
scripts/
  fetch_citations.py  # Queries OpenAlex (fallback: Semantic Scholar)
```

## Key Patterns

### Citation Pipeline
- GitHub Actions runs `scripts/fetch_citations.py` monthly (1st of month)
- Script reads keywords from `data/config.json`, queries OpenAlex API
- Results deduplicated by DOI, filtered, sorted by year desc, written to `data/citations.json`
- `citations.js` renders entries into `#citations-list` on page load

### Theme System
- `theme.js` must load in `<head>` before paint to prevent flash
- Checks `localStorage.theme` first, then `prefers-color-scheme`
- All colors are CSS custom properties in `:root` (light) and `[data-theme="dark"]`

### Scroll Animations
- Elements with `[data-animate]` fade in via IntersectionObserver in `main.js`
- Respects `prefers-reduced-motion: reduce`

## Updating Research Keywords

1. Edit `data/config.json` → `citations.keywords` array
2. Commit and push
3. Optionally trigger Actions workflow manually to see immediate results

## Running Citation Fetch Locally

```bash
pip install requests
python3 scripts/fetch_citations.py
```

Requires `requests` library. Writes to `data/citations.json`.
